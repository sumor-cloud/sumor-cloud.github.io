## التثبيت
```bash
npm install app-deployer -g
```

## المتطلبات الأساسية

### إصدار Node.JS
تتطلب إصدار Node.JS الإصدار 18.x أو أعلى

## الاستخدام

يمكنك استخدام الأمر `ade --help` للحصول على قائمة بجميع الأوامر المتاحة.
```bash
ade --help
```

### نشر

```bash
ade deploy
```

### عرض السجلات

```bash
ade logs
```
