## التثبيت
```bash
npm install app-deployer -g
```

## المتطلبات

### إصدار Node.JS
تتطلب إصدار Node.JS 18.x أو أحدث

### عميل Git
تتطلب [عميل Git](https://git-scm.com/) لاستنساخ المستودع
## الاستخدام

يمكنك استخدام الأمر `ade --help` للحصول على قائمة بجميع الأوامر المتاحة.
```bash
ade --help
```

### النشر

```bash
ade deploy
```

### عرض السجلات

```bash
ade logs
```
