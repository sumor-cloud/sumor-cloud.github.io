## التثبيت

```bash
npm install app-deployer -g
```

## الشروط المسبقة

### إصدار Node.JS

يتطلب إصدار Node.JS 18.x أو أعلى

### عميل Git

يتطلب [عميل Git](https://git-scm.com/) لاستنساخ المستودع

## الاستخدام

يمكنك استخدام الأمر `ade --help` للحصول على قائمة بجميع الأوامر المتاحة.

```bash
ade --help
```

### النشر

```bash
ade deploy
```

### عرض السجلات

```bash
ade logs
```
