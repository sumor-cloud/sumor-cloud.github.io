```bash
npm i @sumor/cloud-app --save
```

## المتطلبات

### إصدار Node.JS
يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب وحدة ES Node.JS
نظرًا لأن هذه الحزمة مكتوبة بوحدة ES،
يرجى تغيير الرمز التالي في ملف `package.json` الخاص بك:
```json
{
    "type": "module"
}
```
