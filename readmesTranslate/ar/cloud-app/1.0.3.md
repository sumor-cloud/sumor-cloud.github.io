## التثبيت
```bash
npm i @sumor/cloud-app --save
```

## الشروط المسبقة

### نسخة Node.JS
تتطلب نسخة Node.JS 16.x أو أعلى

### يتطلب Node.JS استيراد وحدة ES
نظرًا لأن هذه الحزمة مكتوبة بوحدة ES، يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:
```json
{
    "type": "module"
}
```
