## التثبيت

```bash
npm i @sumor/cloud-app --save
```

## المتطلبات الأساسية

### إصدار Node.JS

يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب عنصر Node.JS ES

نظرًا لأن هذه الحزمة مكتوبة بـ عناصر ES،
يرجى تغيير الرمز التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
