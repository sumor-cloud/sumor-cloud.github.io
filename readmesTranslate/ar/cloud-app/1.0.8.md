## التثبيت

```bash
npm i @sumor/cloud-app --save
```

## المتطلبات

### إصدار Node.JS

يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب إصدار Node.JS ES module

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module,
يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
