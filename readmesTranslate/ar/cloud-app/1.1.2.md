## التثبيت

```bash
npm i @sumor/cloud-app --save
```

## المتطلبات الأساسية

### إصدار Node.JS

يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب وحدة Node.JS ES

نظرًا لأن هذه الحزمة مكتوبة بتنسيق وحدة ES،
يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
