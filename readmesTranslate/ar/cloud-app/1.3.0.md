## التثبيت

```bash
npm i @sumor/cloud-app --save
```

## المتطلبات المسبقة

### إصدار Node.JS

يتطلب إصدار Node.JS 16.x أو أعلى.

### يتطلب إصدار Node.JS ES موديول

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES موديول، يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
