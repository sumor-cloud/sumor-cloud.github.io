## التثبيت

```bash
npm i @sumor/cloud-app --save
```

## المتطلبات المسبقة

### إصدار Node.JS

يتطلب إصدار Node.JS 18.x أو أعلى

### يتطلب وحدة إدخال Node.JS ES

نظرًا لأن هذه الحزمة مكتوبة بوحدة إدخال ES،
يرجى تغيير الشيفرة التالية في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
