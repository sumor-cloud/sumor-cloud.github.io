```bash
npm i @sumor/logger --save
```

## المتطلبات
### إصدار Node.JS
يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب نموذجاً ES لـ Node.JS
نظرًا لأن هذه الحزمة مكتوبة بنموذج ES،
يرجى تغيير الكود التالي في ملف ```package.json``` الخاص بك:
```json
{
    "type": "module"
}
```
