```bash
npm i @sumor/short-id --save
```

## المتطلبات الأساسية

### إصدار Node.JS

يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب تحويل ES module لـ Node.JS

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module،
يرجى تغيير الشيفرة التالية في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
