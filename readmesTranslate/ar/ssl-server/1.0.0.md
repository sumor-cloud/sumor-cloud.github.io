# سيرفر ssl
سيرفر الويب SSL مع Express، يدعم HTTP/2

## التثبيت
```bash
npm i @sumor/ssl-server --save
```

## الاستخدام

### استدعاء نموذج ES
يرجى تغيير الكود التالي في ملف "package.json" الخاص بك:
```json
{
    "type": "module"
}
```

### إضافة ملفات SSL
يرجى إضافة ملفات SSL إلى مجلد الجذر ```ssl``` بالأسماء التالية:
- ```domain.crt```
- ```domain.key```
- ```ca.crt```

### استضافة سيرفر بسيط

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

// تستمع على المنفذ 443 افتراضيًا وتعيد توجيه 80 إلى HTTPS 443
await app.listen();
```

## الميزات

### دعم جميع ميزات Express

### إضافة وسيطات ومسارات

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();
const bodyParser = require('body-parser');

// يمكنك إضافة أي وسيط Express
app.use(bodyParser);

// إضافة مسارات
app.get('/', (req, res) => {
    res.send('مرحباً بالعالم!');
});

await app.listen();
```

### إغلاق السيرفر بالقوة

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

await app.listen();
await app.close();
```
