# خادم SSL
خادم ويب SSL مع Express، يدعم HTTP/2

## التثبيت
```bash
npm i @sumor/ssl-server --save
```

## المتطلبات الأساسية

### إصدار Node.JS
يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب Node.JS وحدة ES
نظرًا لأن هذه الحزمة مكتوبة بوحدة ES،
يرجى تغيير الكود التالي في ملف ```package.json``` الخاص بك:
```json
{
    "type": "module"
}
```

## الاستخدام

### استضافة خادم بسيط

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

// استمع على المنفذ 443 بشكل افتراضي، وأعد التوجيه 80 إلى HTTPS 443
await app.listen();
```


### إضافة ملفات SSL
يرجى إضافة ملفات SSL في مجلد الجذر ```ssl``` بالأسماء التالية:
- ```domain.crt```
- ```domain.key```
- ```ca.crt```

إذا لم يتم العثور عليها، سيرفر سيقوم بإنشاء شهادة ذاتية التوقيع.

## الميزات

### دعم جميع ميزات express

### إضافة وسيطات ومسارات

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();
const bodyParser = require('body-parser');

// يمكنك إضافة أي وسيط express
app.use(bodyParser);

// إضافة مسارات
app.get('/', (req, res) => {
    res.send('Hello World!');
});

await app.listen();
```

### إغلاق خادم بالقوة

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

await app.listen();
await app.close();
```
