# ssl-server
خادم الويب SSL مع Express ، يدعم HTTP/2

## التثبيت
```bash
npm i @sumor/ssl-server --save
```

## المتطلبات الأساسية

### إصدار Node.JS
يتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب Node.JS ES module
نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module ،
يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:
```json
{
    "type": "module"
}
```

## الاستخدام

### استضافة خادم بسيط

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

// الاستماع على المنفذ 443 افتراضيًا ، وإعادة توجيه 80 إلى https 443
await app.listen();
```


### إضافة ملفات SSL
يرجى إضافة ملفات SSL إلى مجلد الجذر `ssl` بالأسماء التالية:
- `domain.crt`
- `domain.key`
- `ca.crt`

إذا لم يتم العثور عليها ، سيقوم الخادم بإنشاء شهادة موقع الكتروني ذاتية التوقيع.

## الميزات

### دعم جميع ميزات express

### إضافة middlewares وroutes

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();
const bodyParser = require('body-parser');

// يمكنك إضافة أي middleware من express
app.use(bodyParser);

// إضافة routes
app.get('/', (req, res) => {
    res.send('مرحبًا بالعالم!');
});

await app.listen();
```

### قفل الخادم بقوة

```javascript
const createApp = require('@sumor/ssl-server');
const app = createApp();

await app.listen();
await app.close();
```
