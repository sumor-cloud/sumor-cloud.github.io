## التثبيت

```bash
npm i @sumor/storage --save
```

## الشروط الأساسية

### إصدار Node.JS

تتطلب إصدار Node.JS الحالي 16.x أو أعلى

### يتطلب إصدار Node.JS ES module

نظرًا لكتابة هذه الحزمة باستخدام ES module،
يرجى تغيير الشيفرة التالية في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
