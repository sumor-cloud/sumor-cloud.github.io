## التثبيت

```bash
npm i @sumor/ux --save
```

## المتطلبات الأولية

### نسخة Node.JS

تتطلب نسخة Node.JS الإصدار 18.x أو أعلى

### يتطلب إصدار ES module Node.JS

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module،
يرجى تغيير الشفرة التالية في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
