# تجربة المستخدم

## التثبيت

```bash
npm i @sumor/ux --save
```

## المتطلبات الأساسية

### إصدار Node.JS

تتطلب إصدار Node.JS 16.x أو أعلى

### يتطلب إصدار Node.JS ES module

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module، 
يرجى تغيير الرمز التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
