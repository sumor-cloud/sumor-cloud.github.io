## التثبيت

```bash
npm i @sumor/ux --save
```

## الشروط المسبقة

### نسخة Node.JS

تتطلب نسخة Node.JS 16.x أو أحدث

### تتطلب إصدار Node.JS ES module

نظرًا لأن هذه الحزمة مكتوبة بتنسيق ES module،
يرجى تغيير الشيفرة التالية في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
