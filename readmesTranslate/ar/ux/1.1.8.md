## التثبيت

```bash
npm i @sumor/ux --save
```

## المتطلبات الأساسية

### إصدار Node.JS

يتطلب إصدار Node.JS 18.x أو أعلى

### يتطلب إصدار Node.JS ES module

نظرًا لكتابة هذه الحزمة بتنسيق ES module،
يرجى تغيير الكود التالي في ملف `package.json` الخاص بك:

```json
{
  "type": "module"
}
```
