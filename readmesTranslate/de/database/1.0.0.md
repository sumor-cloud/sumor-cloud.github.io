## Installation

```bash
npm i @sumor/database --save
```

## Voraussetzungen

### Node.JS Version

Erfordert Node.JS Version 16.x oder höher

### Erfordern Node.JS ES-Modul

Da dieses Paket in ES-Modul geschrieben ist,
ändern Sie bitte den folgenden Code in Ihrer `package.json` Datei:

```json
{
  "type": "module"
}
```
