## Installation

```bash
npm i @sumor/ux --save
```

## Voraussetzungen

### Node.JS-Version

Benötigt Node.JS-Version 16.x oder höher

### Erforderliche Node.JS ES-Modul

Da dieses Paket in ES-Modul geschrieben ist,
ändern Sie bitte den folgenden Code in Ihrer `package.json`-Datei:

```json
{
  "type": "module"
}
```
