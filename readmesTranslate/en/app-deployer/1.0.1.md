## Installation

```bash
npm install app-deployer -g
```

## Prerequisites

### Node.JS version

Require Node.JS version 18.x or above

### Git Client

Require [Git Client](https://git-scm.com/) to clone the repository

## Usage

You can use the `ade --help` command to get a list of all available commands.

```bash
ade --help
```

### Deploy

```bash
ade deploy
```

### Display Logs

```bash
ade logs
```
