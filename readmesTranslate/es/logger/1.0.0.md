## Instalación
```bash
npm i @sumor/logger --save
```

## Requisitos previos

### Versión de Node.JS
Requiere Node.JS versión 16.x o superior

### Requerir módulo ES de Node.JS
Como este paquete está escrito en módulo ES,
por favor cambie el siguiente código en su archivo `package.json`:
```json
{
    "type": "module"
}
```
