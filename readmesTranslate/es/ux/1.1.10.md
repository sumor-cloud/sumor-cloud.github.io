## Instalación

```bash
npm i @sumor/ux --save
```

## Requisitos previos

### Versión de Node.JS

Requiere Node.JS versión 18.x o superior.

### Requiere módulo ES de Node.JS

Dado que este paquete está escrito como módulo ES, por favor cambie el siguiente código en su archivo `package.json`:

```json
{
  "type": "module"
}
```
