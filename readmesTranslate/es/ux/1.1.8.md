## Instalación

```bash
npm i @sumor/ux --save
```

## Requisitos previos

### Versión de Node.JS

Requiere una versión de Node.JS 18.x o superior

### Requerir módulo ES de Node.JS

Dado que este paquete está escrito en módulo ES,
por favor cambie el siguiente código en su archivo `package.json`:

```json
{
  "type": "module"
}
```
