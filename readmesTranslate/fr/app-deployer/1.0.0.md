## Installation
```bash
npm install app-deployer -g
```

## Prérequis

### Version de Node.JS
Nécessite Node.JS version 18.x ou supérieure

## Utilisation

Vous pouvez utiliser la commande `ade --help` pour obtenir la liste de toutes les commandes disponibles.
```bash
ade --help
```

### Déploiement

```bash
ade deploy
```

### Afficher les journaux

```bash
ade logs
```
