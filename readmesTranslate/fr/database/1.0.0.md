## Installation

```bash
npm i @sumor/database --save
```

## Prérequis

### Version de Node.JS

Nécessite Node.JS version 16.x ou supérieure

### require Node.JS module ES

Comme ce package est écrit en module ES,
veuillez modifier le code suivant dans votre fichier `package.json` :

```json
{
  "type": "module"
}
```
