## Installation

```bash
npm i @sumor/ux --save
```

## Prérequis

### Version Node.JS

Nécessite la version 18.x ou supérieure de Node.JS

### Requiert un module ES Node.JS

Comme ce package est écrit en module ES,
veuillez modifier le code suivant dans votre fichier `package.json` :

```json
{
  "type": "module"
}
```
