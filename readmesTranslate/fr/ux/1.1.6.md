## Installation

```bash
npm i @sumor/ux --save
```

## Prérequis

### Version de Node.JS

Nécessite la version Node.JS 16.x ou supérieure

### nécessite un module ES de Node.JS

Comme ce package est écrit en module ES,
veuillez modifier le code suivant dans votre fichier `package.json`:

```json
{
  "type": "module"
}
```
