## Installation

```bash
npm i @sumor/ux --save
```

## Prérequis

### Version de Node.JS

Nécessite Node.JS version 18.x ou supérieure

### Nécessite le module ES Node.JS

Comme ce package est écrit en module ES,
veuillez modifier le code suivant dans votre fichier `package.json` :

```json
{
  "type": "module"
}
```
