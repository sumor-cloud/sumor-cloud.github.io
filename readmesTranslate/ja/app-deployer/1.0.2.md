## インストール

```bash
npm install app-deployer -g
```

## 必要条件

### Node.JS バージョン

Node.JS バージョン 18.x 以上が必要です

### Git クライアント

リポジトリのクローンに[Gitクライアント](https://git-scm.com/)が必要です

## 使用法

すべての利用可能なコマンドのリストを取得するには `ade --help` コマンドを使用できます。

```bash
ade --help
```

### デプロイ

```bash
ade deploy
```

### ログの表示

```bash
ade logs
```
