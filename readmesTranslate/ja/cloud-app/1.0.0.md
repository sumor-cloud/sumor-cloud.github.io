## インストール
```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JS バージョン
Node.JS バージョン16.x 以上が必要です

### Node.JS ES モジュールを必要とする
このパッケージはESモジュールで記述されているため、以下のコードをお使いの ```package.json``` ファイルに変更してください:
```json
{
    "type": "module"
}
```
