## インストール
```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JSのバージョン
Node.JSのバージョン16.x以上が必要

### Node.JSのESモジュールの要求
このパッケージはESモジュールで書かれているため、
```package.json```ファイル内の以下のコードを変更してください:
```json
{
    "type": "module"
}
```
