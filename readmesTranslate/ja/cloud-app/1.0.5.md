## インストール
```bash
npm i @sumor/cloud-app --save
```

## 必要なもの

### Node.JSバージョン
Node.JSバージョン16.x以上が必要です

### Node.JS ESモジュールの要件
このパッケージはESモジュールで記述されているため、
次のコードを```package.json```ファイルで変更してください:
```json
{
    "type": "module"
}
```
