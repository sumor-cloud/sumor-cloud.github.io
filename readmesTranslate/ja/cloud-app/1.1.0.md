## インストール

```bash
npm i @sumor/cloud-app --save
```

## 必要条件

### Node.JSのバージョン

Node.JSバージョン16.x以上が必要です

### Node.JSのESモジュールの要件

このパッケージはESモジュールで書かれているため、`package.json`ファイルの中で以下のコードを変更してください:

```json
{
  "type": "module"
}
```
