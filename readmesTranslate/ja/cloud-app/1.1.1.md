## インストール

```bash
npm i @sumor/cloud-app --save
```

## 必要条件

### Node.JS バージョン

Node.JS バージョン 16.x 以上が必要です

### Node.JS ES モジュールの要求

このパッケージは ES モジュールで書かれているため、`package.json` ファイルの次のコードを変更してください:

```json
{
  "type": "module"
}
```
