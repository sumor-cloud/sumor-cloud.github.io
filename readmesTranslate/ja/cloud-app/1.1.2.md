## インストール

```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JS のバージョン

Node.JS のバージョン16.x以上が必要です。

### Node.JS ES モジュールの要件

このパッケージはESモジュールで書かれているため、
`package.json` ファイル内の以下のコードを変更してください:

```json
{
  "type": "module"
}
```
