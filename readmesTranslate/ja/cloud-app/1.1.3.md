## インストール

```bash
npm i @sumor/cloud-app --save
```

## 必要条件

### Node.jsのバージョン

Node.jsのバージョン16.x以上が必要です。

### Node.js ESモジュールの要件

このパッケージはESモジュールで書かれているため、以下のコードを`package.json`ファイルで変更してください:

```json
{
  "type": "module"
}
```
