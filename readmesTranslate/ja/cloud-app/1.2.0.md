## インストール

```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JS バージョン

Node.JS バージョン 16.x 以上を必要とします

### require Node.JS ES モジュール

このパッケージは ES モジュールで書かれているため、  
`package.json` ファイルの以下のコードを変更してください:

```json
{
  "type": "module"
}
```
