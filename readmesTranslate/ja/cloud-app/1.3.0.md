## インストール

```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JSのバージョン

Node.JSバージョン16.x以上が必要です。

### Node.JS ESモジュールの要件

このパッケージはESモジュールで書かれているため、`package.json`ファイル内の次のコードを変更してください:

```json
{
  "type": "module"
}
```
