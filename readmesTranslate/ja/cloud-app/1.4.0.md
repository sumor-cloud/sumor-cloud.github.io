## インストール

```bash
npm i @sumor/cloud-app --save
```

## 必要条件

### Node.JS のバージョン

Node.JS のバージョン 16.x 以上が必要です

### Node.JS の ES モジュールの要件

このパッケージは ES モジュールで書かれているため、お使いの `package.json` ファイル内の以下のコードを変更してください:

```json
{
  "type": "module"
}
```
