## インストール

```bash
npm i @sumor/cloud-app --save
```

## 前提条件

### Node.JS バージョン

Node.JS バージョン 18.x 以降が必要です。

### Node.JS ES モジュールの要件

このパッケージはESモジュールで書かれているため、`package.json`ファイルに以下のコードを変更してください：

```json
{
  "type": "module"
}
```
