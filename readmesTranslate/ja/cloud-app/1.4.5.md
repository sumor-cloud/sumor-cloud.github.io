## インストール

```bash
npm i @sumor/cloud-app --save
```

## 必須条件

### Node.JS バージョン

Node.JS バージョン18.x以上が必要です

### Node.JS ES モジュールが必要

このパッケージはESモジュールで記述されているため、`package.json`ファイル内の以下のコードを変更してください:

```json
{
  "type": "module"
}
```
