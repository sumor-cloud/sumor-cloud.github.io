## インストール

```bash
npm i @sumor/database --save
```

## 必要条件

### Node.JS バージョン

Node.JS バージョン16.x以上が必要です

### Node.JS ESモジュールの要件

このパッケージはESモジュールで書かれているため、次のコードを `package.json` ファイルで変更してください:

```json
{
  "type": "module"
}
```
