## インストール

```bash
npm i @sumor/database --save
```

## 必須事項

### Node.JS バージョン

Node.JS バージョン 16.x 以上が必要です。

### Node.JS のESモジュールの要件

このパッケージは ES モジュールで書かれているため、以下のように `package.json` ファイルを変更してください：

```json
{
  "type": "module"
}
```
