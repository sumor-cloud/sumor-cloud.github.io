## 설치
```bash
npm install app-deployer -g
```

## 사전 요구 사항

### Node.JS 버전
Node.JS 버전 18.x 이상이 필요합니다.

## 사용법

`ade --help` 명령을 사용하여 모든 가능한 명령의 목록을 얻을 수 있습니다.
```bash
ade --help
```

### 배포

```bash
ade deploy
```

### 로그 표시

```bash
ade logs
```
