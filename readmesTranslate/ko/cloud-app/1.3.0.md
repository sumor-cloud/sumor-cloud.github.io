## 설치

```bash
npm i @sumor/cloud-app --save
```

## 준비 사항

### Node.JS 버전

Node.JS 버전 16.x 이상이 필요합니다.

### Node.JS ES 모듈 필요

이 패키지는 ES 모듈로 작성되어 있으므로,
`package.json` 파일에서 다음 코드를 변경해주셔야 합니다:

```json
{
  "type": "module"
}
```
