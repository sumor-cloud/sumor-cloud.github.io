## 설치

```bash
npm i @sumor/database --save
```

## 전제 조건

### Node.JS 버전

Node.JS 버전 16.x 이상이 필요합니다.

### Node.JS ES 모듈 필요

이 패키지는 ES 모듈로 작성되었기 때문에,
`package.json` 파일에서 다음 코드를 변경해 주세요:

```json
{
  "type": "module"
}
```
