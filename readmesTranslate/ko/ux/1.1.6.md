[Sumor Cloud](https://sumor.cloud) 도구입니다.

## 설치

```bash
npm i @sumor/ux --save
```

## 전제 조건

### Node.JS 버전

Node.JS 버전 16.x 이상이 필요합니다.

### Node.JS ES 모듈 요구 사항

이 패키지는 ES 모듈로 작성되었으므로, `package.json` 파일에서 다음 코드를 변경해야 합니다:

```json
{
  "type": "module"
}
```
