[Sumor Cloud](https://sumor.cloud) 도구.

## 설치

```bash
npm i @sumor/ux --save
```

## 필수 조건

### Node.JS 버전

Node.JS 버전 18.x 이상이 필요합니다.

### Node.JS ES 모듈이 필요합니다

이 패키지는 ES 모듈로 작성되었으므로,
`package.json` 파일에서 다음 코드를 변경해주세요:

```json
{
  "type": "module"
}
```
