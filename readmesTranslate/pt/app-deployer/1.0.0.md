## Instalação
```bash
npm install app-deployer -g
```

## Pré-requisitos

### Versão do Node.JS 
Requer Node.JS versão 18.x ou superior

## Uso

Você pode usar o comando `ade --help` para obter uma lista de todos os comandos disponíveis.
```bash
ade --help
```

### Implantação

```bash
ade deploy
```

### Exibir Logs

```bash
ade logs
```
