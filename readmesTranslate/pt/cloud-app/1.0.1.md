## Instalação
```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão do Node.JS
Requer a versão do Node.JS 16.x ou superior

### Requer módulo ES do Node.JS
Como este pacote é escrito em módulo ES, por favor altere o seguinte código no seu arquivo `package.json`:
```json
{
    "type": "module"
}
```
