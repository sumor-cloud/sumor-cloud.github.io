## Instalação

```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão do Node.JS

Requer a versão Node.JS 16.x ou superior

### Requer módulo ES do Node.JS

Como esse pacote está escrito em módulo ES,
por favor altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
