## Instalação

```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão do Node.JS

Requer Node.JS versão 16.x ou superior

### Requerir módulo ES do Node.JS

Como este pacote é escrito em módulo ES, por favor altere o código a seguir em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
