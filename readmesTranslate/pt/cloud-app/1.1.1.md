## Instalação

```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão do Node.JS

Requer Node.JS na versão 16.x ou superior

### Requer módulo ES do Node.JS

Como este pacote foi escrito em módulo ES,
por favor altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
