## Instalação

```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão Node.JS

Requer a versão Node.JS 16.x ou superior

### Requer módulo ES do Node.JS 

Como este pacote é escrito em módulo ES,
altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
