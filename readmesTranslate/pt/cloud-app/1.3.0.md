## Instalação

```bash
npm i @sumor/cloud-app --save
```

## Pré-requisitos

### Versão Node.JS

Requer Node.JS versão 16.x ou superior

### Requerir Módulo ES Node.JS

Como este pacote está escrito em módulo ES,
por favor altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
