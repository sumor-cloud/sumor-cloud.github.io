## Instalação

```bash
npm i @sumor/database --save
```

## Pré-requisitos

### Versão Node.JS

Requer Node.JS versão 16.x ou superior

### requer módulo ES do Node.JS

Como este pacote é escrito em módulo ES,
por favor altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
