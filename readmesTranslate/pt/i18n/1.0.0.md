# i18n

Um [Sumor Cloud](https://sumor.cloud) Tool.  
[Mais Documentação](https://sumor.cloud)

Este é uma biblioteca i18n leve para Node.js e o navegador.
Você pode facilmente usá-lo para gerenciar seus recursos i18n.
E aplicá-lo ao seu projeto.

## Instalação

```bash
npm i @sumor/i18n --save
```

## Pré-requisitos

### Versão Node.JS

Requer Node.JS versão 16.x ou superior

### Requer ES module do Node.JS

Como este pacote é escrito em módulo ES,
por favor altere o seguinte código no seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
