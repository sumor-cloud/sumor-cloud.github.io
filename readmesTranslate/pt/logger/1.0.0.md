## Instalação
```bash
npm i @sumor/logger --save
```

## Pré-requisitos

### Versão do Node.JS
Requer Node.JS versão 16.x ou superior

### Necessário `require` de Módulo ES do Node.JS
Como este pacote foi escrito em Módulo ES, altere o seguinte código em seu arquivo ```package.json```:
```json
{
    "type": "module"
}
```
