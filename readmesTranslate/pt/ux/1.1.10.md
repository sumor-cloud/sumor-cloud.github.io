## Instalação

```bash
npm i @sumor/ux --save
```

## Pré-requisitos

### Versão Node.JS

Requer Node.JS versão 18.x ou superior

### Requerir Módulo ES Node.JS

Como este pacote é escrito em módulo ES,
por favor altere o seguinte código em seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
