## Instalação

```bash
npm i @sumor/ux --save
```

## Pré-requisitos

### Versão do Node.JS

Requer Node.JS versão 18.x ou superior

### Requerir Node.JS ES module

Como este pacote é escrito em módulo ES,
por favor, altere o seguinte código no seu arquivo `package.json`:

```json
{
  "type": "module"
}
```
