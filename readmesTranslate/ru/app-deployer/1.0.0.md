## Установка

```bash
npm install app-deployer -g
```

## Предварительные требования

### Версия Node.JS

Требуется версия Node.JS 18.x или выше

## Использование

Вы можете использовать команду `ade --help`, чтобы получить список всех доступных команд.

```bash
ade --help
```

### Развернуть

```bash
ade deploy
```

### Отображение журналов

```bash
ade logs
```
