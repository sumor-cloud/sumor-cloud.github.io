## Установка

```bash
npm i @sumor/cloud-app --save
```

## Требования

### Версия Node.JS

Требуется версия Node.JS 16.x или выше.

### Требуется модуль ES в Node.JS

Поскольку этот пакет написан как модуль ES,  
пожалуйста, измените следующий код в вашем файле `package.json`:

```json
{
  "type": "module"
}
```
