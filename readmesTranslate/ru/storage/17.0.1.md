# @sumor/storage
轻云存储框架

## 安装

通过 [npm](https://www.npmjs.com/) 安装:
```sh
$ npm install --save @sumor/storage
```

## 使用说明

ES6方式
```js
//例如要访问的程序路径为level1.level2.level3
import {level1} from '@sumor/storage';

level1.level2.level3('parameter1');
```
Node.js CommonJS方式
```js
//例如要访问的程序路径为level1.level2.level3
const {level1} = require('@sumor/storage');

level1.level2.level3('parameter1');
```
