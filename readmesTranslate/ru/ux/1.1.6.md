## Установка

```bash
npm i @sumor/ux --save
```

## Предварительные требования

### Версия Node.JS

Требуется версия Node.JS 16.x или выше

### Требуется модуль Node.JS ES

Так как этот пакет написан на ES-модулях,
пожалуйста, измените следующий код в вашем файле `package.json`:

```json
{
  "type": "module"
}
```
