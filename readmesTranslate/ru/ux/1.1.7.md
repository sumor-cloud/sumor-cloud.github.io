## Установка

```bash
npm i @sumor/ux --save
```

## Предварительные требования

### Версия Node.JS

Требуется версия Node.JS 16.x или выше

### Требуется Node.JS ES модуль

Поскольку этот пакет написан в ES модуле,
пожалуйста, измените следующий код в вашем файле `package.json`:

```json
{
  "type": "module"
}
```
