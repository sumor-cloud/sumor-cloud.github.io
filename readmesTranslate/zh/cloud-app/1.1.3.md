## 安装

```bash
npm i @sumor/cloud-app --save
```

## 先决条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或更高

### 需要 Node.JS ES 模块

由于此包是使用 ES 模块编写的，请在您的 `package.json` 文件中更改以下代码：

```json
{
  "type": "module"
}
```
