## 安装

```bash
npm i @sumor/cloud-app --save
```

## 先决条件

### Node.JS 版本

需求 Node.JS 版本为 18.x 或以上

### 需要 Node.JS ES 模块

由于此包是用 ES 模块编写的，请在您的 `package.json` 文件中更改以下代码：

```json
{
  "type": "module"
}
```
