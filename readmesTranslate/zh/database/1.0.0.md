## 安装

```bash
npm i @sumor/database --save
```

## 先决条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或以上

### 需要 Node.JS 的 ES 模块

由于此包是用 ES 模块编写的，请在您的 `package.json` 文件中修改以下代码：

```json
{
  "type": "module"
}
```
