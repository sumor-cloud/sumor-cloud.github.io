## 安装

```bash
npm i @sumor/database --save
```

## 前提条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或更高

### 需要 Node.JS ES 模块

由于该软件包是以 ES 模块编写的，请更改您的 `package.json` 文件中的以下代码：

```json
{
  "type": "module"
}
```
