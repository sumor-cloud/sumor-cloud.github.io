# i18n

一个[Sumor Cloud](https://sumor.cloud)工具。  
[更多文档](https://sumor.cloud)

这是一个轻量级的用于 Node.js 和浏览器的 i18n 库。  
您可以轻松使用它来管理您的 i18n 资源。  
并将其应用到您的项目中。

## 安装

```bash
npm i @sumor/i18n --save
```

## 先决条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或更高

### 需要 Node.JS ES 模块

由于该包是使用 ES 模块编写的，请在您的 `package.json` 文件中更改以下代码：

```json
{
  "type": "module"
}
```
