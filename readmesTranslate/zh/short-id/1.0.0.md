# Short-ID

一个[Sumor Cloud](https://sumor.cloud)工具。  
[更多文档](https://sumor.cloud)

这是一个供 Node.js 和浏览器使用的 short-id 库。
您可以轻松地使用它从数字生成一个短 ID。

## 安装

```bash
npm i @sumor/short-id --save
```

## 先决条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或以上

### 需要 Node.JS ES 模块

由于这个包是用 ES 模块编写的，
请将以下代码添加到您的 `package.json` 文件中：

```json
{
  "type": "module"
}
```
