# 存储

一个 [Sumor Cloud](https://sumor.cloud) 工具。  
[更多文档](https://sumor.cloud/storage)

这是一个适用于 Node.JS 的轻量级存储库。  
它可以连接到各种存储服务，比如阿里云 OSS

## 安装

```bash
npm i @sumor/storage --save
```

## 先决条件

### Node.JS 版本

需要 Node.JS 版本 16.x 或更高版本

### 需要 Node.JS ES 模块

由于此软件包是用 ES 模块编写的，  
请更改您的 `package.json` 文件中的以下代码:

```json
{
  "type": "module"
}
```
