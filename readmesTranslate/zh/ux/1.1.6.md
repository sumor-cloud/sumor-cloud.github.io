## 安装

```bash
npm i @sumor/ux --save
```

## 先决条件

### Node.JS版本

需要Node.JS版本16.x或更高版本

### 需要Node.JS ES模块

由于此软件包是用ES模块编写的，请在您的`package.json`文件中更改以下代码：

```json
{
  "type": "module"
}
```
